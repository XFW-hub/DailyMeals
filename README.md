# 每日三餐记录

记录每天三餐、零食、夜宵，支持五档评级与图片上传，可多人使用。

## 功能

- **餐次类型**：早餐、午餐、晚餐、零食、夜宵
- **五档评级**：夯 / 顶级 / 人上人 / NPC / 拉完了
- **图片上传**：每条记录可附带一张图片（可选）
- **多用户**：添加多个用户，按用户筛选记录
- **按日期筛选**：查看某一天的记录

## 使用前准备

1. 安装 [Node.js](https://nodejs.org/)（建议 16+）
2. 在项目目录打开终端

## 启动方式

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

浏览器访问：**http://localhost:3000**

## 使用步骤

1. 点击「+ 添加用户」创建用户，或在顶部选择已有用户
2. 选择要记录的日期（默认今天）
3. 在「添加记录」中选择类型（早餐/午餐/晚餐/零食/夜宵）、填写吃了什么、选择评级，可选上传图片
4. 点击「保存记录」
5. 下方列表会按用户和日期显示记录，可删除单条记录

## 数据与图片

- 用户与记录保存在项目内 `data/` 目录（`users.json`、`records.json`）
- 上传的图片保存在 `public/uploads/`
- 未部署到服务器时，数据仅在本机有效；多人使用可部署到同一台机器或内网，大家用同一地址访问

## 技术说明

- 后端：Node.js + Express
- 前端：原生 HTML/CSS/JS
- 图片：multer 本地存储
