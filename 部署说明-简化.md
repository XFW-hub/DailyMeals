# 用 Glitch 部署（免卡、最简单）

不用绑卡、不用配 Nix，从 GitHub 导入就能跑。

---

## 步骤（约 2 分钟）

### 1. 打开 Glitch

浏览器打开：**https://glitch.com**  
用 **GitHub** 登录（或注册）。

---

### 2. 从 GitHub 导入项目

- 点 **New project**（或 **Create new project**）
- 选 **Import from GitHub**
- 在输入框里粘贴你的仓库地址：
  ```
  https://github.com/XFW-hub/DailyMeals
  ```
- 点 **Import**，等一会儿导入完成。

---

### 3. 等它自动启动

Glitch 会识别这是 Node 项目，自动执行 `npm install` 和 `npm start`。  
左侧有 **Logs** 的话可以看一眼，出现 “三餐记录服务已启动” 就说明跑起来了。

---

### 4. 拿到你的网页链接

- 点左上角 **Share**（或 **Preview**）
- 选 **Live site** 或 **Open in new tab**
- 会得到一个地址，类似：**https://xxxx.glitch.me**
- 这个链接就是你的公网地址，发给自己或别人都能打开用。

---

## 小提示

- **休眠**：免费版一段时间没人访问会休眠，第一次打开可能要等几秒。
- **改代码**：在 Glitch 里改完会自动重启；若想用本机改，改完推到 GitHub 后，在 Glitch 里点 **Tools → Import and export → Import from GitHub** 再导入一次即可更新。

---

不用 Replit、不用 Render 绑卡，就这一种方式即可。
